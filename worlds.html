World Generator

The world shown here is generated randomly using a procedural-generation technique that borrows concepts from cosmology in order to create terrain on the surface of a sphere. A basic erosion model gives realistic relief to mountains. The spherical world is perfectly periodic, meaning that continents and oceans line up automatically without seams. The API used to generate these worlds is hosted on Google Cloud Run.
